-> Overflow: Desbordacion cuando se tiene la cantidad permitida en un nodo. 
se produce cuando se quiere agregar una clave a un nodo el cual ya tiene la cantidad máxima de claves permitidas.


-> Underflow: se produce cuando se quiere eliminar una clave de un nodo el cual ya tiene la cantidad mínima de claves permitidas.

-> Redistribución: La redistribución sucede cuando un nodo tiene underflow, se puede trasladar una llave de un hermano adyacenente a este nodo, para que el underflow deje de ocurrir.

-> Fusión/Concatenacion: Si un nodo hermano esta al minimo y no se puede redistribuir, se concatena con un nodo adyacente disminuyendo la cantidad de nodos, y en algunos casos la altura del árbol.
